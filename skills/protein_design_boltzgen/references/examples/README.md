# Bundled Preset Examples

These example inputs mirror the 7 BoltzGen presets available in the Composer UI.

Preset folders:

- `vanilla_target_binding_site`
- `vanilla_protein`
- `binding_disordered_peptides`
- `protein_binding_small_molecule`
- `small_molecule_from_file_and_smiles`
- `cyclic_against_hiv_antibody_site`
- `nanobody_against_penguinpox_multi_spec`

Supporting multi-spec scaffold folder:

- `nanobody_scaffolds`

Use these files directly when users ask for a ready-to-run example.
